# AI Incident Response Runbook

A guided build for the one document you need before you need it: a tailored
AI data-exposure runbook with clear owners, first steps, and a communication
path for the first 72 hours.

Built by MBL Partners (https://mblpartners.com) — part of the Thrive with AI
skill library.

---

## What It Does

Walks you through four intake questions to tailor the runbook to your
organization:

1. Tools in scope — Which AI tools could receive sensitive data?
2. Data classes — What categories of data would be most damaging if exposed?
3. Owner — Who is the named person responsible when an exposure happens?
4. Obligations — Are you under regulatory or contractual notification requirements?

From those answers, it produces a complete, organization-specific runbook
covering three response phases: Assess, Contain, and Communicate. It also
outputs the three steps you need to have memorized before an incident occurs.

An AI exposure is different from a breach. A breach is a door someone forced.
An exposure is something your own people handed over. The runbook is built
around those questions: what was given away, to whom, under what terms, and
what you can still claw back.

## Install

1. Download `incident-response-runbook.skill`
2. Open Cowork
3. Drag the `.skill` file into your Cowork window
4. The skill installs automatically and is ready to run

---

## How to Run It

Once installed, trigger it by saying any of the following:
- "Build an incident runbook"
- "AI incident response plan"
- "What do we do if data leaks?"
- "Incident runbook"

The skill asks one question at a time and waits for your response before moving
on. Run it on a calm day — that is the only day it is easy.

---

## Output

A tailored runbook containing:
- Phase 1: Assess — what was exposed, the tool's retention terms, and scope
- Phase 2: Contain — revoking access, requesting deletion, and documentation
- Phase 3: Communicate — internal notification path and external obligations
- The three steps to memorize before an incident
- A contacts list with named owner, legal, security, and key vendor channels
